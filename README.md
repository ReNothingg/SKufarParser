# СKufarParser

Телеграм-бот для мониторинга новых объявлений на Kufar в выбранном регионе/районе.

## Что уже реализовано

- Фоновый мониторинг новых объявлений.
- Выбор региона и района через inline-кнопки.
- Команда `/all` для просмотра объявлений списком.
- Показ всех фото объявления (до 10 изображений).
- Baseline-механика: при смене региона старые объявления помечаются как уже просмотренные, чтобы не было спама.

## Быстрый старт

1. Создать и активировать виртуальное окружение.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

1. Создать `.env` на основе шаблона:

```bash
cp .env.example .env
```

1. Заполнить переменные в `.env`.
2. Запустить бота:

```bash
python main.py
```

## Переменные окружения

- `BOT_TOKEN` - токен Telegram-бота (обязательно).
- `USER_ID` - Telegram user ID, куда отправлять уведомления о новых объявлениях (обязательно).
- `CHECK_INTERVAL` - интервал проверки новых объявлений в секундах (по умолчанию `60`).
- `LOCATIONS_FILE` - путь к файлу локаций (по умолчанию `locations.json`).
- `KUFAR_AUTH_TOKEN` - **опциональный** токен для запросов Kufar. Можно без него.
- `KUFAR_USER_AGENT` - user-agent для HTTP-запросов.
